Include images in your wordpress RSS.

== Description ==

Simple plugin that allows to include the images into your wordpress rss, rss2.
The plugin use the post/page thumbnail or the firt attached image to the post/page gallery. viste the <a href="http://web-argument.com/2008/11/09/wp-rss-images-wordpress-plugin/">plugin page</a> for details.

== Installation ==

1. Upload ‘wp rss images’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress
3. Go to Settings/WP RSS Images and select the image size (Thumbnail, Medium, Full) and feed type (rss, rss2).
